# Clothing Store Frontend

A modern React-based frontend for an online clothing store.

## Features

- **Product Management**: Browse, filter, and view detailed product information
- **Shopping Cart**: Add, remove, and update quantities of products in cart
- **Checkout Process**: Complete purchases with integrated backend API
- **Admin Dashboard**: Manage products, stock levels, and view reports
- **User Authentication**: Login, registration, and role-based access control
- **Theme Support**: Light and dark mode toggle
- **Responsive Design**: Mobile-friendly interface

## Tech Stack

- **React 18** with TypeScript
- **React Router 6** for navigation
- **Styled Components** for styling
- **Context API** for state management
- **Axios** for API communication
